Fixing missing Document Properties dialog in Inkscape on Mac OS X¶

It has been bugging me for a while that I could not open the Document Properties dialog on Inkscape on my mac. The button clicks, but the window would not show. I noticed that the main Inkscape window would lose focus, but alas no properties dialog.

I finally decided to do something about it today. It turns out that you have to edit the preferences file and manually give x and y coordinates to this dialog. I found the answer here and here.

In ~/.config/inkscape/preferences.xml, find

<group
    id="documentoptions"
    state="1" />

and change it by adding coordinatesL

<group
    id="documentoptions"
    x="500"
    y="500"
    state="1" />

The bug is apparently due to poor multi-monitor support, but anyway, there ya go!
